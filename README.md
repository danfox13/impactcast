### Launching the Server and Client
To launch the **server**, use `npm run server` inside `server` directory.

To launch the **client**, use `npm start` inside `client` directory.

### Things to Consider
For the sake of keeping the repository tidy, **do not** push files/folders generated by the IDE or node.js (e.g. `.idea/`, `node_modules/`).
